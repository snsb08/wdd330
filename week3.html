JavaScript: Novice to Ninja

Chapter 5 Objects

Objects are like a database of values or a dictionary.
Objects are assigned by reference. If two variables reference the same object, they will both change when one is modified.
JSON is both human and machine readable.

  1. Property names must be double-quoted
  2. Permitted values are double-quoted strings, numbers, true, false, null, arrays and objects
  3. Functions are not permitted values

Rounding Methods

Math.ceil() to round up
Math.floor() to round down
Math.round() to round to the nearest integer
Math.trunc() to cut the decimal point (gets truncated)

Maximum & Minimum Methods

Math.max() returns the maximum number
Math.min() returns the minimum number
Be careful with rounding numbers if the result needs to be exact because it is often inaccurate. Computers have lots of trouble dealing with decimal fractions and the results can vary depending on the web browser or operating system. But most of the time rounding errors are not a problem.

To generate a random number between 0 and another number, multiply Math.random() by that number.
The Epoch is a date used in programming as a reference point in time to measure dates. It is 1st January 1970.

RegExp = Regular expression that use a pattern to find strings.
Preferred method: Using literal notation between forward slashes:
const pattern = /[a-zA-Z]+ing$/;
Alternative: Using new operator and a constructor function:
const pattern = new RegExp('[a-zA-Z]+ing');

Question/Comment: I wonder if people use regExp for family history search. Since many times names are misspelled by just one letter.

Some string methods accept regular expressions as a parameter.
For example:
'JavaScript'.match(/[aeiou]/g); // return an array of all the vowels
<< ['a', 'a', 'i']
'JavaScript'.replace(/[aeiou]/ig,'*');
<< 'J*v*Scr*pt'

Chapter 6 The Document Object Model

The DOM is a document that has elements connected like branches and nodes of a plant or tree.
The DOM is not part of JavaScript. DOM can be used with any programming language, but JavaScript is the language most commonly used with it.

Getting elements

There are different methods to access the elements of a page so they can be assigned to variables and be inspected or modified.

document.getElementById() -an element with a unique Id name. Only one can have that name.
document.getElementsByClassName() -multiple elements can share a class name.
document.querySelector()
Dot notation

The attributes of an element can be modified or added using setAttribute. It takes two arguments: the attribute that you wish to change, and the new value of that attribute.

Elements can be created with createElement().
Child elements can be modify with appendChild(), replaceChild(), or removeChild().

Chapter 7 Events

The addEventListener() can help to add some functions that run in the background and let us know when they happen. They only start if the event argument is true and if not, the program keeps moving forward.

It is better to avoid events inside the HTML code. Keep JavaScript code separately, out of the HTML.
Using the name Event makes the code more readable but it is not mandatory to call it Event. Many developers use the "e" for short.
Be careful about using a single click and double-click event in the same element because it can get confusing and the double-click might trigger the single click event too.
It is important to add both mouse events and touch events because of the variety of devices that can be used.
The swipe effect is done with a combination of touchstart and touchmove.
